I would read off the color of LED that needed to blink and the length of the note, and Annie would write the code. Of course I understood the code and I could've written it myself, but it was more efficient this way. The problems are evident: Annie had to do all the grunt work. I don't think anything was good about my teamwork. I tried to be accommodating to Annie's choices, including the hyper-precision necessary to line the LEDs up with the song. (We calculated delays sometimes down to hundredths or thousandths of a millisecond.)

The basic parts of an Arduino sketch are the setup and the loop functions. In the setup you can initialize everything and write all the code that only needs to be executed once. In the loop function you can write everything that needs to be repeated forever, or at least until the program stops.

MVC Questions

An MVC paradigm is a pattern used to build applications. The M stands for model, the part that stores the data. The controller (C) handles user input, and the view (V) is the user interface -- what the user sees.

Frameworks are a reusable design that handles common tasks so, for example, you don't have to suffer through the same procedures 6,000,000 other programmers before you did just to make a website, you can get straight to customizing.

A library is a specific collection of files and code, while a framework is anything you use to make your project.
